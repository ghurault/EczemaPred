# Markov Chain expected transition matrix

Markov Chain expected transition matrix

## Usage

``` r
plot_transition_MC(fit, max_scale = NA, show_text = FALSE)
```

## Arguments

- fit:

  Stanfit object corresponding to the Markov Chain model

- max_scale:

  Maximum value that the legend display. If NA, this chosen
  automatically.

- show_text:

  Whether to display the probability as text in the heatmap.

## Value

Ggplot
